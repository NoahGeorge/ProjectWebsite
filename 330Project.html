EE 330 Final Project

Noah George

In my junior year, I took EE 330. This class covered p-mos and n-mos transistors and how to use them to create filters and amplifiing circuits. For the final project in this class, the professor let us choose between an analog and digital project. I am interested in hardware design, so I chose the digital project: Creating and synthesizing a digital calculator.

The project was set up in groups of three people. Our calculator had to be able to add, subtract, multiply, and divide a three digit decimal number with another one digit decimal number, and output the answer decoded to be displayed on a seven-segment display in decimal. The inputs were in binary. There were two main stages of the project: creating the calculator design, and synthesizing it to create the actual circuit. Because of the requirement of the software we were using for synthesis, the design had to be created in Verilog. Because of my focus on computer engineering and my experience in my internships with VHDL, I worked on creating the design with one teammate, while the other working on figuring out synthesis.

I had briefly used Verilog for another class my sophomore year, but I had never used it to create a project of this scale. I was excited to have the opportunity to learn the syntax and build experience with it. Through this project I also gained knowledge of synthesis tools that create circuits from digital designs.
